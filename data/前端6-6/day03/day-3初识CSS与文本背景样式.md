## 什么是css?

```
Cascading Style Sheets (层叠样式表)
是一种用来为html文档加样式(字体 间距 位置 颜色 角度等等)的计算机语言

css以html为基础 提供了丰富的功能 如字体 颜色 背景的控制以及整体排版等等
```


## CSS选择器

```
css的工作分为两步 1.找到对象 2.加样式
选择器就是找到模板的手段 通过选择器找到一个指定的标签元素

1.标签选择器: 选中一个标签 一般用来清除默认样式
list-style: none; 清除列表默认样式
	语法格式:
		标签名{
			属性:值;
			属性:值;
		}
		 p{
            color: #096;
            font-size: 20px;
        }
        
2.id选择器:给标签设置id编码 然后通过#id名的方式找到这个标签
id名一定要是唯一性 单个标签不能有多个id名
    语法格式:
    <标签名 id="id名">
            #id名{
                属性:值;
                属性:值;
            }

3.类选择器(class) 给标签设置class类型 然后通过.类名的方式找到对应标签 类可以多个 id只能一个 
	语法格式:
    <标签名 class="类名">
            .类名{
                属性:值;
                属性:值;
            }
            
4.通配符选择器(*) 表示选中网页里的所有元素 (覆盖面广 选中所有标签 性能造成浪费 一般开发中不使用 一般情况下我们页面的标签样式不会长得一样)       语法格式:
		*{
			属性:值;
			属性:值;
		}
```

## CSS书写位置

```
css代码分为:
内部样式
外部样式
行内样式
```

```
1.内部样式表 --> 把样式表卸载当前页面里的style标签中 style写在head标签里
	语法格式:
        <head>
            <style>
				css代码
            </style>
        </head>
        
2.外部样式表 --> 创建外部样式表文件 后缀使用.css 通过link标签引入外部样式表
	rel="引入文件资源类型" href="引入文件资源的路径"
	<link rel="stylesheet" href="./demo.css">
	
3.行内样式表 --> 通过style属性 把样式之间写在标签里
	 <标签名 style="属性: 值; 属性:值;">

样式表优先级就近原则 -- 行内样式 > 内部样式/外部样式(谁距离标签最近听谁的)
样式表总结:
	1.行内样式表:
		优点: 写起来比较方便,权重比较高
		缺点: 灵活性不强 只能改一个
		频率: 较少
		范围: 单个标签
	2.内部样式表
		优点: 写起来方便,在单个页面时比较灵活
		缺点: 可复用性不强
		频率: 较多(学习阶段用它为主)
		范围: 整个网页
	3.外部样式表
		优点: 可复用性高
		缺点: 需要导入
		频率: 最高(正式开发时.大部分是用link)
		范围: 整个网站
```

## div盒子标签(division)

```
div本身是没有含义 他是一般作为划分区域的标签 他是一个很单纯的标签
可塑性很强 一般是用来作为容器 所以称为盒子 把对应的内容写在div盒子里
div本身是没有宽高/颜色属性
```

```

快速创建一个class为xx的盒子 .xxx
```

## 背景样式(background)

```
background:背景
	background-color:设置背景颜色
	background-image:设置背景图片
	
	
背景图片和img标签使用
如果为了给盒子加底色内容部分加背景 或者是加一些小图标项目符号等 则使用背景图片 如果是想表达一个图片信息 则使用Img标签
background-position: x y;设置图片位置
	x轴 left左边 center居中 right右边
	y轴 top上  center居中 bottom下
	
background-size:x y;设置图片大小
	cover 等比例缩放 (盒子多大 图片多大)
	
background-attachment:设置背景图片是否随着滚动轴滚动	
	background-attachment: fixed;背景随着滚动而滚动
	background-attachment: local;背景和内容都滚动
	background-attachment: scroll;默认值 背景不随着滚动而滚动
	
background-repeat:设置背景图片是否重复
	background-repeat: no-repeat;不重复
	background-repeat: repeat-x; 水平平铺
	background-repeat: repeat-y; 垂直平铺
	background-repeat: repeat;默认
```

## 注意:

```
命名:
	1.见名知意
	2.文件和元素命名全英文小写 禁止驼峰(MyName  MyName) 多个单词建议使用'-'连字符top-bar
	3.不允许命名带有广告等英文单词 防止模块被浏览器当做垃圾广告过滤 (文件命名如此)

1.写选择器时,要遵循一定的顺序:  通配符 > 标签选择器 > 类/id选择器

2.标签选择器一般不是用来'加样式的' 而是用来'清除样式'的 (很多标签都有自带的样式.a标签默认蓝色+下划线.如果想取消就用a标签) 比较常见的是清除列表样式/内外边距/字体大小.  标签选择器一般不是用来单独加特效.而是设置默认值
3.属性值后面要有分号
```
